# 福祉サポーターズサポートアプリ・そよぎ（高齢者版）

介護・医療現場の**従事者（スタッフ）向け**の会話サポートアプリ。若手スタッフが高齢の利用者・患者と
スムーズに会話・回想（回想法）を行うための「お守り」。開発：介護と支援の相談どころ そよぎ。

- **主目的はスタッフの手元での参照**。利用者に画面を見せる場面もあり得るので、見せても差し支えない作りにしている。
- 個人情報の入力機能は持たない（法的リスク回避）。素材は CC0／絵文字中心。

## 3タブ構成
1. **今日のネタ** … 今日の花＋花言葉／今日は何の日（明るい出来事）。挨拶・雑談の糸口。
2. **回想クイズ** … ジャンル別クイズ。答え合わせ後に**豆知識**と、会話へ橋渡しする**★スタッフ用カンペ**を表示。
   「肩ならし（大小・重さ）」は認知機能が低下した方でも直感で答えられる設問。
3. **アイスブレイク** … 「どっち派？」と「色で気分診断（結果は必ず前向き）」。

## 設計方針
- **スコア／レベルは無し**。ゲームではなく会話ツール。不正解を赤バツで突きつけず、やさしく正解を開示する。
- **技術**：素の HTML/CSS/JS + PWA（オフライン動作）。Farcaster Mini App・Google Play（Capacitor）へ展開。
  Flutter を使わないのは Farcaster がWebベース前提のため（[[proj-farcaster-mini-apps]] と同方針）。

## 多言語・多文化（重要）
UI文言は `i18n.js`（翻訳可能）。**会話の中身は翻訳できない**ため `content.<code>.js` に
**文化ごとに別物**として収録する。対象は「高齢者が多い×高齢者福祉が整備された先進国」の7か国：

| code | 国 | code | 国 |
|------|----|------|----|
| ja | 日本 | it | イタリア |
| en | アメリカ（英語圏） | es | スペイン |
| de | ドイツ | ko | 韓国 |
| fr | フランス | | |

## ファイル
- `index.html` … アプリ本体（3タブ・エンジン）
- `i18n.js` … UI文言（言語追加は langs+ui に1ブロック）
- `content.<code>.js` … 文化別コンテンツ（花／今日は何の日／回想クイズ／アイスブレイク）。スキーマは `content.ja.js` 冒頭コメント参照。
- `manifest.webmanifest` / `sw.js` … PWA
- `.well-known/farcaster.json` … Farcaster（デプロイ後に署名）
- `vercel.json` … Vercel配信設定
- `icons/` … アイコン（`icon.svg` あり。PNG各サイズは後で生成）

## 言語の増やし方（3ステップ）
1. `i18n.js` の `langs` に `{code, native, dir}` を追加
2. `i18n.js` の `ui` に該当言語ブロックを追加（`en` をコピーして訳す）
3. `content.<code>.js` を作成（※**翻訳ではなく**その国の記憶で新規著述）＋ `sw.js` の ASSETS に追加
